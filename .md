2.2 RESTful API Architecture

REST (Representational State Transfer) is a software architectural style used for building scalable web services.

Key Principles:

Stateless communication

Client-server separation

Uniform resource identification

HTTP method usage (GET, POST, PUT, DELETE)

StreamBeat follows REST architecture using Django REST Framework.

2.3 JWT Authentication

JSON Web Tokens (JWT) are used for secure stateless authentication.

Benefits:

Secure token-based login

No server-side session storage

Scalable for distributed systems
